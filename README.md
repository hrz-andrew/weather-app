# 🌊 Bay Area Weather Dashboard

A personal weather dashboard for comparing microclimates across Bay Area freshwater fishing locations.

**Live:** https://hrz-andrew.github.io/weather-app/

---

## What it does

Side-by-side 6-day forecast for 5 freshwater and 3 saltwater Bay Area spots. Compares temperature, wind, rain, humidity, pressure, UV index, and cloud base across locations so you can pick the best day and spot.

## Models

Switch between ECMWF, GFS, and NAM via the model selector in the header.

## Tech

Vanilla HTML/CSS/JS · Chart.js · Open-Meteo API · GitHub Pages
